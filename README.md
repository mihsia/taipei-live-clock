# 臺灣標準時間即時鐘

這是一個適合嵌入 PowerPoint 的 16:9 即時時鐘頁面，固定顯示台北時間、24 小時制、中華民國日期、西元日期與星期。

公開網址：

```text
https://mihsia.github.io/taipei-live-clock/
```

## PowerPoint 嵌入建議

1. 先用本機或線上伺服器提供此資料夾，例如：

   ```bash
   python3 -m http.server 8765
   ```

2. 在 PowerPoint 使用 Web Viewer / 網頁嵌入工具，填入：

   ```text
   http://localhost:8765/
   ```

3. 建議投影片使用寬螢幕 16:9，並讓網頁物件填滿投影片。頁面已保留安全邊界，避免文字或翻頁卡片貼齊邊緣。

4. 若要提供給其他電腦或投影設備使用，請將頁面部署到可連線的網址，再把 PowerPoint 內嵌網址改成該線上網址。

## 更新紀錄

- 建立 16:9 靜態即時鐘頁面，固定使用 Asia/Taipei 時區，顯示中華民國與西元日期、星期、24 小時制。
- 調整為 PowerPoint 友善版面，保留投影片安全邊界，避免 Web Viewer 內嵌時產生捲軸或裁切。
- 將時間分成 HOUR / MIN / SEC 三組，提高時、分、秒判讀性。
- 發布至 GitHub Pages，提供可長期使用的公開網址。
- 依展示需求優化視覺層級：日期縮小，時分為主要焦點，秒數降階呈現。
- 加入宜蘭．臺灣、Taiwan Standard Time、UTC+8 等在地化資訊。
- 加入深色科技背景、玻璃感面板、翻頁卡高光、邊緣光與陰影。
- 調整為 Midnight Slate 配色：深藍灰底、青綠點綴、沉穩現代。
- 針對 Mac 版 PowerPoint 修正字型渲染差異，避免粗體數字在翻頁卡內被裁切。
- 加入冒號分隔符，讓時間更接近 HH:MM:SS 的閱讀節奏。
- 加深翻頁牌中間橫線，強化機械翻頁感。
- 加大牌面圓角，使整體視覺更柔和。
- 放大 HOUR / MIN / SEC 標籤以提升遠距閱讀性。
